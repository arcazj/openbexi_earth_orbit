# OpenBEXI Solar System Ephemeris

This folder contains local runtime ephemeris data for OpenBEXI Earth Orbit Version 1.7.

## Dataset

- File: `solar_system_jpl_horizons_2020_2035_6h.json`
- Reference samples: `solar_system_jpl_horizons_reference_samples.json`
- Source: NASA/JPL Horizons API
- Generated by: `tools/generate_jpl_ephemeris.py`
- Runtime network use: none
- Supported date range: `2020-01-01T00:00:00Z` through `2035-12-31T00:00:00Z`
- Sample cadence: 6 hours
- Interpolation: linear position interpolation between 6-hour samples
- Reference frame: ICRF
- Reference plane: ecliptic
- Origin: Sun center, Horizons center `500@10`
- Units: kilometers and kilometers per second
- Time handling: UTC timestamps from Horizons calendar output; runtime interpolation uses UTC milliseconds

## Accuracy Targets

- Mercury, Venus, Earth, Moon, and Mars: less than `5,000 km` against off-grid Horizons validation samples.
- Jupiter, Saturn, and Uranus: less than `50,000 km` against off-grid Horizons validation samples.

The bundled validation samples are queried independently from Horizons at off-grid times so tests measure interpolation error rather than exact table lookup.

## Limitations

This is a JPL-derived visualization ephemeris, not a certified flight-dynamics or navigation product. Do not use it for spacecraft operations, conjunction assessment, or safety-critical analysis without independent validation.

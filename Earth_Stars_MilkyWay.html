Data and limitations

Bundled demo stars use real J2000/ICRS-style RA/Dec, magnitude, and B-V color-index values for bright and selected nearby stars. The demo is intentionally small so this page runs offline from the repo.

Magnitude <18 is not loaded directly. Gaia DR3 contains about 1.8 billion sources; a browser page must use external tiled/LOD/binary chunks for that depth. For a practical local browser view, use magnitude <10 by default and Tycho-2-like data up to about <11.5.

The page attempts to load obj/Textures/starmap-4k.jpg as a local Milky Way sky texture. If unavailable, it creates a procedural equirectangular Milky Way texture. For production, replace it with a documented CC/public-domain sky map.

Useful source references: ESA Gaia DR3, NASA HEASARC Tycho-2, and the optional Solar System Scope / Wikimedia Milky Way texture licensed CC BY 4.0.

Coordinate conversion is fixed-sphere J2000/ICRS RA/Dec. Proper motion, parallax, precession, nutation, and observer topocentric corrections are not applied here. Add astronomy-engine only if epoch/time transforms are later required.
